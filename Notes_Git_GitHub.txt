Navegação via Command Line

💻 GUI X CLI
💻 Git é um software com design voltado para outro tipo de programa
💻 CLI – Command Line Interface – interagir por linha de comando
💻 GUI – Graphical User Interface
💻 CMD – Terminal
💻 Flags: /S /Q


Comandos básicos para um bom desempenho no terminal

💻 Listar – dir
💻 Navegar entre as pastas - cd/
💻 Retroceder - cd..
💻 Limpar o terminal – cls
💻 TAB – autocompleta nomes
💻 Criar pasta/diretório – mkdir nome_pasta
💻 Printa o que você digitar- echo
💻 > joga em um arquivo
💻 C:\workspace>echo hello > hello.txt
💻 Deletar – del workspace (limpa todos os arquivos do diretório/pasta workspace)
💻 Remover repositório/pasta (remove directory) – rmdir workspace 
💻  git add .  ou  git add -A (Para adicionar todos os arquivos novos ao contole de versão local)
💻 -m (colocar uma menssagem)
💻 $ git commit -m "Inclusão das anotações do cursoGit/GitHub" (versionamento, commit para o GitHub)
💻 Após o arquivo estar preparado para commit ➡️ $ git push origin main (envia todos os commits locais para a Nuvem)
